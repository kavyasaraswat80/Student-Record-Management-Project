# Student Record Management Project

## Overview

This project demonstrates database building and management using MS Excel.

## Features

* Managed records of 500 students
* Organized student data using spreadsheets
* Applied formulas for result calculations
* Used sorting and filtering for efficient data handling
* Categorized student information department-wise

## Tools Used

* MS Excel
* MS Office

## Skills Demonstrated

* Database Management
* Data Entry
* Data Organization
* Spreadsheet Handling
* Record Management
